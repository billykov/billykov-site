Connected Board Navigator

How to Use

Connected Board Navigator adds a Navigate tab to your item views and a shortcut to the row action menu — giving you one-click access to any board connected through a Connect Boards column. No searching, no copying names.

Installation

Go to the monday.com Marketplace and install Connected Board Navigator. That's it — the Navigate connections shortcut is immediately available in the ... action menu on any item.

Using the Action Menu Shortcut

For quick navigation without opening the full item:

  • Hover over any item row on the board
  • Click the ... menu
  • Select Navigate connections
  • The navigation panel opens instantly, showing all connected items grouped by column

Adding the Navigate Tab (optional)

For a persistent navigation view inside the item view:

  • Open any item on a board that has Connect Boards columns
  • In the item view tab bar, click + Add view
  • Find Connected Board Navigator in the list — if you don't see it, click Explore more views and search for it
  • The Navigate tab is now added — you only need to do this once per board

Navigating to a Connected Item

  • Open any item and click the Navigate tab, or use the ... action menu shortcut
  • All connected items are listed, grouped by column
  • Click Open → next to any item to jump directly to it on its board

Free vs Pro

  • Free — full functionality on 1 board, no time limit
  • Pro — unlimited boards, $5/month or $4/month billed yearly
